### Log in

<details>
<summary>To install the Otterize CLI</summary>

```bash
otterize login
```

### Create an integration for your cluster
<Tabs groupId="cli-ui">
<TabItem value="cloud-ui" label="Cloud UI" default>

:::danger
Not sure if it's better to use images or plain text.
:::
* Select the **Integrations** tab.
* Create a new *Kuberenetes* integration with the <div class="add-an-integration-h2">Add an integration</div> button.
* Follow the connection instruction for your integration by clicking the <div class="connect-button-h2">Connect</div> button.
[//]: # (![Otterize Cloud - network mapper]&#40;/img/getting-started/integrations-tab.png&#41;)
</TabItem>
<TabItem value="cli" label="CLI" default>

:::danger
Pending CLI support
:::
Create a cluster
```bash
otterize clusters create ...
```
Create an integration for the cluster
```bash
otterize integrations create ...
```

We will use the *client id* and *client secret* in the next step.

</TabItem>
</Tabs>

### Integrate your cluster
<Tabs groupId="cli-ui">
<TabItem value="cloud-ui" label="Cloud UI" default>

</TabItem>

Upgrade/install your Otterize OSS deployment using the *client id* and *client secret* generated for your integration.
